值班长周末短信业务量攻略

KPI指标上报攻略 发布人：金培

一、 集团话务网管

http://10.1.85.112:8000/webpm_new/default.aspx
登录名shanghai 密码Sh@12345
报表查看/考核报表/中国移动集团GPRS报表/GPRS性能报表，网元类型选“省”，时间粒度选“日报”

全天Gb流量（单位:TB）=（SGSN GB上行流量+SGSN GB下行流量）/1024/1024
CMNET地市入流量(TB)=(流出平均速率(Mbps)/8/1024/1024*3600*24)
集团网管URL：http://10.1.122.139:18080/dnms/login.do
用户名/密码：shanghai/Shanghai2014
打开集团网管，进入首页，点击左侧的业务管理。点击“业务报表（新框架）”，展开“部省接口CMNET”，点击“省内中继流量统计报表”，选择省份和时间，即可查询省内中继天粒度数据。

二、 LTE指标攻略

自动取法： http://10.221.187.233/lte.html 查看“LTE日报”表格，一般在每天6:35左右生成，请注意查看数据的日期 LTE吞吐量=全天空口流量TB=(上行吞吐量（空口）+下行吞吐量（空口）)/1024
如上述页面无数据，请自行从话务网管取报表。攻略如下：

 MME附着用户数：
- 话务网管-测试报表-集团报表-TDLTE网络性能指标详情表 - 取“省”粒度，“小时”粒度，前一天24小时
- 取“MME附着用户数”字段，本地一般取10点数值，集团可能全天最大值/10000
 上下行吞吐量 - 话务网管-测试报表-集团报表-TDLTE网络性能指标详情表
- 取“省”粒度，“小时”粒度，前一天24小时
- 取“上行吞吐量(空口)”和“下行吞吐量(空口)”字段，全天求和，单位GB
 无线接通率 - 话务网管-测试报表-集团报表-TDLTE网络性能指标详情表
- 取“省”粒度，“小时”粒度，前一天24小时。六忙时取8、9、10、18、19、20点
- 无线接通率 = ( SUM(RRC连接建立成功次数)*SUM(E-RAB建立成功数)) / (SUM(RRC连接建立请求次数)*SUM(E-RAB建立请求数))
 无线掉线率 - 话务网管-测试报表-集团报表-TDLTE网络性能指标详情表
- 取“省”粒度，“小时”粒度，前一天24小时。六忙时取8、9、10、18、19、20点
- 无线掉线率 = (SUM(eNB请求释放上下文数)-SUM(正常的eNB请求释放上下文数)) / SUM(初始上下文建立成功次数)  排除用户原因的附着成功率
- 话务网管-测试报表-集团报表-LTE最新报表-MME网元性能报表
- 取“省”粒度，“小时”粒度，前一天24小时。六忙时取8、9、10、18、19、20点
- 取“排除用户原因的附着成功率”字段，6个时段算数平均（话务网管报表目前没有分子分母）
